SGBD (Sistema Gerenciador de Banco de Dados)
Tipos(
	Relacionamantos entre tabelas
		Exemplos não relacionais: DBU, Twitch, Excel
		Exemplos relacionais: MySQL, Oracle, SQLServer, PostGree, MongoDB, Access.

	Linguagem
		Exemplos de SGBD SQL: MySQL, Oracle, SQLServer, PostGree, Access.
		Exemplos de SGBD não SQL: MongoDB (JSON), DBU(CSV)
)

Estrutura de dados
TXT, CSV, XML, JSON
SQL “Structured Query Language”

Banco de dados:
Estrutura Base -> Tabela -> Linhas e Colunas
Tabela -> Colunas = Campos
Tabela -> Linhas = Registros (Dados)

Banco de Dados Relacional
Chaves (Primária x Estrangeira)
Chave Primária ou PK = Um campo que não se repete e serve de índice para um relacionamento entre tabelas
Chave Estrangeira ou FK = Um campo que normalmente se repete e serve de índice para um relacionamento entre tabelas

Modelo:
Modelo Conceitual
Modelo Lógico ou Relacional
Entidade -> 1 ou maisTabela
Atributos -> 1 ou mais Campos ou até uma tabela
Tuplas -> 1 ou mais Registros
